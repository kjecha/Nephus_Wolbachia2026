# Nephus_Wolbachia2026
This repository contains the R code file used for data visualization and analysis from the manuscript ["Wolbachia facilitates the reproduction of a parthenogenetic ladybug"](https://www.biorxiv.org/content/10.64898/2026.06.16.732365v1) -Jecha et al., 2026  
  
Data can be found on figshare: 10.6084/m9.figshare.32803286
